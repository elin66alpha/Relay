# AgentDeck 开发记录与后续规划待办

本文件记录了当前已完成的 UI 提升工作，以及下一步规划的待办事项（关于抽屉栏和设置项的进一步简化与扩展）。

---

## 一、已完成的 UI/UX 改造工作

经过最近的迭代，我们对 `AgentDeck` 客户端进行了深度清理和现代化改造：

### 1. 抽屉栏净化与“设置”集成
- **侧边抽屉栏净化**：移除了原侧边栏底部杂乱的语言切换、白天黑夜切换、额度查询、清空工作目录、扫描二维码等入口。
- **全新“设置”入口**：在抽屉栏底部新增了一个统一的“设置” ListTile，点击跳转至独立的 `AppSettingsScreen`。
- **外观与语言管理**：在设置界面中，以卡片化、高颜值的 Material 3 `SegmentedButton` 呈现了**外观设置**（白天/黑夜/跟随系统）与**语言设置**（中文/英文）。

### 2. 机器状态栏交互升级（红绿在线指示灯）
- **状态栏入口化**：抽屉栏顶部原本静态显示的机器名称现在转变成一个**高亮可点击的互动卡片（ActiveMachineStatusTile）**。
- **红绿状态指示**：将原本的云朵图标替换为极简、直观的状态圆点图标（`Icons.lens`）。
  - **绿色圆点 (`#10B981`)**：代表机器**在线**。
  - **红色圆点 (`#EF4444`)**：代表机器**离线/未在线**。
  - 配合本地化文案，分别显示为 `在线` 或 `未在线`。
- **弹窗显示详细状态**：点击该卡片会始终弹出当前机器的详细连接诊断信息（支持 Mono 字体显示错误堆栈）。
- **实时刷新功能**：状态弹窗内集成了“刷新”按钮，结合 `StatefulBuilder`，点击可在弹窗内即时重新校验连接，并同步更新抽屉栏卡片的红绿指示状态。
- **避免冗余**：已从独立的“设置”页面中彻底拉出机器连接状态校验逻辑。

### 3. AppBar 右上角聊天操作重构
- **图标对调**：将右上角的 **清空当前对话** (`Icons.refresh_rounded`) 与 **额度查询** (`Icons.query_stats_outlined`) 位置对调，将高频的清空按钮移至左侧。
- **新增“压缩”动作**：在清空当前对话图标的右边，新增了一个 **压缩** 图标（`Icons.compress`）。
- **发送压缩指令**：点击压缩图标可以直接向 AI Agent 发送特殊的 `/compact` 指令，调用后端的会话压缩逻辑，以精简上下文。

### 4. 国际化与自动化编译部署
- **本地化文案**：在 `app_strings.dart` 中新增了 `online`、`offline`、`compress` 三个中英文键值，保证多语言流畅切换。
- **静态代码检查**：通过了 `flutter analyze` 100% 无错误/警告验证。
- **打包与实机部署**：成功执行 `flutter build apk` 打包 release，并通过 `adb install -r` 成功推送到连接的测试设备上，完成了发布与实机验证。

## 二、已完成的后续事项

根据最新设计方案，以下事项已经落地到代码：

### 1. 抽屉栏去掉“机器”列表段落（彻底去冗余）✅
- **修改文件**：`lib/features/cli_agents/cli_agents_drawer.dart`
- **具体改动**：
  - 在 `ListView` 中，找到并彻底移除 **“机器” (context.l10n.machines)** 这一小节标题（`Text` 标题 widget）。
  - 彻底移除紧随其后的机器凭证列表循环渲染代码：
    ```dart
    for (final MachineCredential machine in machinesController.credentials)
      ListTile(
        leading: Icon(...),
        title: Text(machine.displayName),
        subtitle: Text(machine.hostLabel),
        ...
      )
    ```
- **改动理由**：
  - 抽屉栏顶部已经有了交互式的“状态显示机器”（即 `ActiveMachineStatusTile`），显示了当前激活的机器名称、连接状态点以及提供详细状态弹窗。
  - 原有的“机器”列表直接展示所有导入的机器名和公网 IP/域名，与顶部卡片以及“管理凭证”界面的功能严重重合，且容易暴露敏感的公网 IP 信息。
  - 去掉此列表后，侧边栏能够大幅缩短，UI 更加清爽高级，符合极简设计美学。

### 2. 设置界面增加“关于”入口（About Screen）✅
- **修改文件**：`lib/features/settings/app_settings_screen.dart`
- **具体改动**：
  - 在 `ListView` 的底部，新辟一个“关于”区域，包含小标题 `context.l10n.about`（需要在 `app_strings.dart` 中添加多语言翻译）。
  - 新增一个关于卡片或 `ListTile`，点击可弹窗或跳转二级页面：
    - **显示应用名称**：`AgentDeck` / `智能体工作台`。
    - **显示版本号**：当前为 `Version 0.1.0+1`。
    - **显示版权与许可信息**。
    - 可以加入后端版本的读取或技术支持联系方式。
- **改动理由**：
  - 提供关于入口是商业化/高级移动端应用的必备组件，能帮助用户快速确认当前的 App 编译版本以及许可条款，提升整体产品的信任度与完整性。
